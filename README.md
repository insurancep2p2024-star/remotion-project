# 🎬 Remotion Project

Video project built with [Remotion](https://www.remotion.dev/).

## 🚀 Getting Started

```bash
npm install
npm start
```

This opens the Remotion Studio at `http://localhost:3000`.

## 🎥 Render Video

```bash
npm run render
```

Output will be saved to `out/video.mp4`.

## 📁 Project Structure

```
src/
  index.ts          # Entry point — registers compositions
  Root.tsx          # Root component with all compositions
  MyComposition.tsx # Main video composition
```

## 🛠 Built With

- [Remotion](https://www.remotion.dev/) v4
- React 18
- TypeScript
